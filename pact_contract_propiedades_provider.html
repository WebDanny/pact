A pact between propiedades_provider and propiedades_provider

Requests from propiedades_provider to propiedades_provider

  • A request for propiedades given there are no propiedades

  • A request for propiedades given there are propiedades

Interactions

Given there are no propiedades, upon receiving a request for propiedades from propiedades_provider, with

{
  "method": "GET",
  "path": "/propiedades"
}

propiedades_provider will respond with:

{
  "status": 200,
  "headers": {
    "Content-Type": [
      "application/json"
    ]
  },
  "body": {
    "content": [

    ],
    "contentType": "application/json",
    "encoded": false
  }
}

Given there are propiedades, upon receiving a request for propiedades from propiedades_provider, with

{
  "method": "GET",
  "path": "/propiedades"
}

propiedades_provider will respond with:

{
  "status": 200,
  "headers": {
    "Content-Type": [
      "application/json"
    ]
  },
  "body": {
    "content": [
      {
        "id": 1234567890,
        "items": [
          {
            "estado": "string",
            "id": "string",
            "nombre": "string",
            "precio": 100
          }
        ]
      }
    ],
    "contentType": "application/json",
    "encoded": false
  }
}
